To sync with your crew, you need a free Firebase database. One person sets this up, everyone else just gets the link.
One-time setup (60 seconds):
1. Go to console.firebase.google.com
2. Click Create a project → name it anything → skip analytics
3. In the left sidebar, click Build → Realtime Database
4. Click Create Database → pick any location → choose Start in test mode
5. Copy the URL at the top (looks like https://your-project-default-rtdb.firebaseio.com)
Paste your database URL
Your URL is saved in this browser so you won't need to enter it again.
